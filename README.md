# GIT-Club
GIT cheat sheet

```
git init
git remote add origin git@github.com:<user>/<repo>.git
git remote -v
git pull origin master
```

```
git init
git remote add origin <url>
git add .
git commit -m "Initial commit"
git push -u origin master
```
